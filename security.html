Security Policy

Last updated: 2026-06-13

NexusPanel is designed to connect to real IT systems, so security matters.

Users should protect their accounts, use strong passwords, keep integration keys private, and only connect systems they are authorized to manage.

Agent keys, API keys, client secrets, database credentials, and service account files should never be shared publicly.

If you believe you found a security issue, report it privately instead of posting it publicly.
